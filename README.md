# FUTURE_DS_02
# Customer Retention & Churn Analysis

## Overview

This project analyzes customer churn and retention patterns using the Telco Customer Churn dataset. The analysis identifies key churn drivers and provides business recommendations to improve customer retention.

---

## Tools Used

- Python (Pandas)
- Tableau Public
- Microsoft Excel
- GitHub

---

## Objectives

- Clean and prepare customer data
- Analyze customer churn patterns
- Identify key retention drivers
- Compare churn across customer segments
- Build an interactive Tableau dashboard
- Generate business recommendations

---

## Key Insights

- Churn Rate: **26.58%**
- Retention Rate: **73.42%**
- Customers with month-to-month contracts showed the highest churn.
- Long-term customers demonstrated stronger retention.
- Electronic check users experienced higher churn.
- Customers with higher monthly charges were more likely to leave.

---

## Business Recommendations

- Promote long-term contracts.
- Improve customer onboarding.
- Encourage automatic payment methods.
- Review pricing strategies.
- Focus retention efforts on new customers.
